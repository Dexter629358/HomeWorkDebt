# Task 4

Given a string path, which is an absolute path (starting with a slash '/') to a file
or directory in a Unix-style file system, convert it to the simplified canonical
path.

## Example

- `/home/` -> `/home`
- `/../` -> `/`
- `/home//foo/` -> `/home/foo`
